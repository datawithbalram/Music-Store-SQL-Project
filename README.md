# Music Store SQL Project

## Project Overview

This project analyzes a Music Store Database using SQL.

The goal of this project is to solve real-world business problems using SQL queries and gain hands-on experience with data analysis.

## Key Insights

### Customer Analysis
- Identified the highest spending customer in the database.
- Determined which city generated the highest revenue.
- Analyzed customer purchasing behavior through invoice data.

### Sales Analysis
- Identified countries with the highest number of invoices.
- Found the highest-value invoices in the dataset.
- Evaluated revenue contribution from different customer groups.

### Music Analysis
- Identified Rock music listeners.
- Determined the top Rock artists based on track count.
- Found tracks longer than the average song duration.

### Advanced Analytics
- Used CTEs and Window Functions for ranking and analytical queries.
- Performed genre-wise and customer-wise analysis.
- Applied business-oriented SQL techniques to derive meaningful insights.
