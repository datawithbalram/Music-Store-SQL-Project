# Music Store SQL Project

## Project Overview

This project analyzes a Music Store Database using SQL.

The goal of this project is to solve real-world business problems using SQL queries and gain hands-on experience with data analysis.

---

## Skills Demonstrated

- SQL Joins
- Aggregate Functions
- GROUP BY
- ORDER BY
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Business Data Analysis

---

## Database Schema

![Database Schema](schemas%20for%20table.png)

The database contains information about:

- Customers
- Employees
- Invoices
- Invoice Lines
- Artists
- Albums
- Tracks
- Genres
- Playlists

---

## Questions Solved

### Easy Level
- Senior most employee
- Country with most invoices
- Top invoice totals
- Best customer city
- Best customer

### Moderate Level
- Rock music listeners
- Top rock artists
- Songs longer than average duration

### Advanced Level
- Customer spending analysis
- Genre popularity analysis
- Revenue-based ranking problems
